# $state

## Информация

::: tip `$state`

- **$state** - руна для реактивного состояния: при изменении значения Svelte заново обрабатывает зависимости и обновляет DOM
  :::

## Структура

- Обычная переменная `let count = 0` при присвоении не триггерит обновление
- `$state(0)` создаёт реактивную ячейку: при присвоении `count += 1` все места, где читается `count`, получают новое значение и интерфейс перерисовывается

```tsx
<script lang="ts">
  let count = $state(0)

  const increment = () => {
    count += 1
  }
</script>

<button onclick={increment}>
  Count is {count}
</button>
```
