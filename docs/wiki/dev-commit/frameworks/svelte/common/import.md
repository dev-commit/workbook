# Импорт из .svelte файла

### Информация

- Отдельный module-блок нужен чтобы **экспортировать** тип из .svelte-файла.
- В Svelte обычный `<script>` — это экземпляр компонента: его типы и переменные наружу не отдаются, их нельзя импортировать из другого файла.
- Чтобы тип жил в одном месте и при этом не нужен был module в компоненте, его можно вынести в отдельный .ts и импортировать и в нужном месте

### Пример

```html
<script lang="ts" module>
  // Тип на экспорт
  export type Model = {
    id: string;
    title: string;
  };
</script>

<script lang="ts">
  // Использование типа
  const model: Model = {};

  // Код для компонента
</script>

<div>Компонент</div>
```
