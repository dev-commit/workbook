```

> **Custom containers** - инструмент VitePress

## Описание

````
> **Описание**
> ```css
> div {
>   shape-outside: ellipse(rx ry at x y);
> }
> ```
> - `rx` - радиус по оси "x" (по умолчанию 50%)
> - `ry` - радиус по оси "y" (по умолчанию 50%)
