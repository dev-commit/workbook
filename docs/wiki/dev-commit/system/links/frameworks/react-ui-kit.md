# React Ui-библиотеки

## Готовые компоненты

- https://mantine.dev/ - Mantine UI (можно использовать и как headless)
- https://chakra-ui.com/ - Chakra UI
- https://www.heroui.com/ - HeroUI
- https://park-ui.com/ - Park UI (основана на Ark UI, связан с Chakra UI)
- https://vercel.com/geist/i - Vercel Geist Design System (стильно, не так много компонентов)
- https://mui.com/ - Material UI (заточена под CSS-in-JS)
- https://ant.design/ - Ant Design (были проблемы с производительностью)
- https://www.radix-ui.com/ - Radix UI (раздел “themes” - стилизованный компоненты, раздел “primitives” - unstyled)

## Headless (unstyled)

> Стили добавляются самостоятельно

- https://ark-ui.com/ - Ark UI (много интересных компонентов, разработан командой Chakra UI)
- https://base-ui.com/ - Base-UI (от Material-UI)
- https://headlessui.com/ - Headless UI (от Tailwind, мало компонентов)
- https://primereact.org/ - Prime React

## Прочее

- https://ui.shadcn.com/ - UI Shadcn (компоненты не ставятся как npm-зависимость - нужно скопировать исходный код и править)
- https://www.shadcn-form.com/ - Shadcn Form (для форм)
- https://carbondesignsystem.com/ - Carbon Design System
- https://reactstrap.github.io/ - React Strap (Bootstrap for React)
- https://www.ninna-ui.dev/ - Ninna UI (компоненты для Tailwind)
