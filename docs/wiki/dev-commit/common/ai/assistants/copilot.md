## Функционал

- Текущий файл: автодополнение кода, объяснение, fix кода и т.д.
- Генерация тестов
- Создание файлов и разворачивание проекта
- Указание контекста
- Справка по API библиотек
- Вопросы по VSCode
- Работа с PR GitHub

### Элементы

::: details Панель Copilot Chat
**Информация**

- Может работать только с 1 файлом

**Команды**

- `@workspace ТекстЗапроса` - можно задать вопрос по всему проекту
- `@workspace /new ТекстЗапроса` - н-р: "Создай новый проект на React". Появляется кнопка "Create workspace...", после нажатия на кнопку создается директория и файлы
- `@vscode ТекстЗапроса` - можно задать вопрос по VSCode. Н-р: “Как изменить тему?”
- `@vscode /search ТекстЗапроса` - Умный поиск: Copilot добавляет Regex в поиск. Н-р: “js files without imports”
  :::

::: details Панель Copilot Edits
**Информация**

- Может создавать файлы
- Можно работать с несколькими файлами
- Нужно явно указать файлы с которыми нужно работать (не нашел как добавить директорию). Copilot Edits не вносит изменений за пределами рабочего набора, за исключением случаев, когда предлагается создать новый файл
  :::

::: details Editor

**Автодополнения кода**

1. Начать вводить код
2. Copilot предложит варианты продолжения

**Комментарий и Ghost Text**

1. Написать комментарий
2. Нажать `Enter`, предложит код функции (Ghost Text)
3. Нажать `Tab` - подставится код
4. Нажать `Enter` и повторить процесс

**GitHub Copilot Suggestions**

1. Написать комментарий
2. Нажать `Ctrl` + `Enter`
3. Откроется окно GitHub Copilot Suggestions, в котором будут разные предложения от Copilot

**Inline Chat**

1. Нажать `Ctrl` + `I`
2. Откроется окно "Ask Copilot"
3. Ввести запрос
4. Код будут написан в текущем документе под запрос

:::

::: details Terminal CLI

**Функционал**

- Можно задавать вопросы по Терминалу

**Алгоритм работы**

1. Нажать `Ctrl` + `I`
2. Откроется окно для запроса. Ввести команду и нажать "Run"
3. Может создавать файлы
   :::
