# MCP

## Информация

::: tip Model Context Protocol (MCP)

- **Model Context Protocol (MCP)** - для интеграции внешних инструментов. Подключается к базам данных, API и источникам документации
  :::

## MCP серверы

- https://context7.com/ - context7 (использование актуальных версий библиотек)
- https://www.hyperbrowser.ai/ - hyperbrowser (web-скрапинг)
- Chrome MCP (тестирование приложения в браузере)
- Figma
