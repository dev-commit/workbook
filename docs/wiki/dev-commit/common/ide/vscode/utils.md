# Инструменты

## Встроенный браузер

- Встроенный **Integrated Browser** в preview.
- Умеет открывать страницы прямо во вкладке редактора, поддерживает localhost, авторизацию и DevTools.
- Открывается через команду

<v-breadcrumbs keys :items="['Ctrl', 'Shift', 'P']" />

```bash
> Browser: Open Integrated Browser
```
