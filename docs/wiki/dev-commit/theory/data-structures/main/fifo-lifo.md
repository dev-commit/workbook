# Методы FIFO и LIFO

## Информация

::: tip Определения

- **FIFO** (First In First Out) - «первый пришёл — первый вышел». Удалить элемент можно только после того, как были убраны все ранее добавленные элементы

- **LIFO** (Last In First Out, «последним пришёл — первым вышел») - последний элемент, добавленный в стек, первым выйдет из стека
  :::

## Примеры использования

- **FIFO** применяется в очередях (например, очередь в банке)
- **LIFO** используется в стеках (например, откат действий в текстовом редакторе)

## Характеристика

- **FIFO** обеспечивает порядок обработки элементов в том же порядке, в котором они были добавлены
- **LIFO** позволяет быстро возвращать последнее добавленное значение, что полезно для операций отката

## Примеры реализации:

- **FIFO** можно реализовать с помощью `Queue` (очередь)
- **LIFO** реализуется через `Stack` (стек)
