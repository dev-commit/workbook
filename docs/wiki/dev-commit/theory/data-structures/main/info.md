::: tip Структуры данных

- **Структуры данных** - способы хранить и организовывать множество однотипных и/или логически связанных данных
- Набор имплементаций (алгоритмов). Структура данных работает с памяться с помощью интерфеса

  :::

- Структуры данных позволяют производить _4 основных типа действий_: доступ, поиск, вставку и удаление
- _Назначение структур данных_: представление системы в удобном для работы вид

::: tip Определения

- **Алгоритм** - имплементация действия
- **Интерфейс** - это методы. Каждый метод использует свой алгоритм чтобы работать с памятью

  :::
