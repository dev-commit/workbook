# CSS Multi Columns

## CSS-свойства

::: details `column-count` - кол-во колонок

```css
article {
  column-count: 2;
}
```

:::

::: details `column-width` - ширина колонки

```css
article {
  column-width: 200px;
}
```

:::

::: details `column-gap` - расстояние между колонками

```css
article {
  column-gap: 20px;
}
```

:::

::: details `column-rule` - линия между колонками

```css
article {
  column-rule: 1px dotted #000;
}
```

:::

## Дополнительные свойства

- `column-fill`
- `column-span`
- `column-rule-color`
- `column-rule-style`
- `column-rule-width`
