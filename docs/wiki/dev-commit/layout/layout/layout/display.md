# display

| Значение       | Описание                         |
| -------------- | -------------------------------- |
| `block`        | Блочный элемент                  |
| `inline`       | Строчный элемент                 |
| `inline-block` | Блочно-строчный элемент          |
| `table`        | Блочная таблица                  |
| `table-cell`   | Ячейка таблицы                   |
| `none`         | Элемент удален из блочной модели |
| `flex`         | Flexbox контейнер                |
| `grid`         | Grid контейнер                   |

```css
div {
  display: block;
  display: inline;
  display: inline-block;
  display: table;
  display: table-cell;
  display: none;
  display: flex;
  display: grid;
}
```
