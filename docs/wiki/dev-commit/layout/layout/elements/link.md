# Ссылки

## HTML-теги

::: details `<a />` - ссылка

```html
<a href="https://ya.ru/">Текст для ссылки</a>
```

**Атрибут `"target"`**

- Значение "\_blank" открывает ссылку в новой вкладке

```html
<a href="page.html" target="_blank">Открытие ссылки в новой вкладке</a>
```

:::

## Виды ссылок

**Ссылки на страницы текущего сайта и на внешние ресурсы**

```html
<a href="page.html">Ссылка на другую страницу</a>
<a href="https://ya.ru/">Ссылка на другой сайт</a>
```

**Якорные ссылки**

- Ссылка секцию текущей страницы

```html
<a href="#about">Ссылка секцию текущей страницы</a>
```

**Ссылки на телефон и e-mail**

```html
<a href="tel:7777777">777-777</a> <a href="mailto:my@mail.com">my@mail.com</a>
```

**Ссылка на скачивание файла**

```html
<a href="test.rar" download>Скачать файл</a>
```
