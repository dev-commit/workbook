# image-set()

## CSS-функции

### `image-set()`

::: tip `image-set()`

- **image-set()** - - Функция позволяет указать список изображений, которые браузер сможет выбрать в качестве фонового изображения, основываясь на том, что он знает о возможностях своего дисплея и скорости соединения. Это аналогично тому, что делает свойство `srcset`
  :::

```css
div {
  background-image: image-set(
    "image.png" 1x,
    "image-2x.png" 2x,
    "image-print.png" 600dpi
  );
}
```
