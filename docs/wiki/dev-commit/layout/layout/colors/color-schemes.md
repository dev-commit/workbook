## Базовые цветовые схемы

### Hexadecimal Colors (HEX)

::: tip Hexadecimal Colors

- **Hexadecimal Colors (Шестнадцатеричные цвета) -** шестнадцатеричные целые числа определяют компоненты цвета. Все значения должны находиться в диапазоне от 00 до FF
  :::

::: details Таблица HEX цветов

| **Интенсивность** | **Значение в HEX** |
| ----------------- | ------------------ |
| 100%              | FF                 |
| 95%               | F2                 |
| 90%               | E6                 |
| 85%               | D9                 |
| 80%               | CC                 |
| 75%               | BF                 |
| 70%               | B3                 |
| 65%               | A6                 |
| 60%               | 99                 |
| 55%               | 8C                 |
| 50%               | 80                 |
| 45%               | 73                 |
| 40%               | 66                 |
| 35%               | 59                 |
| 30%               | 4D                 |
| 25%               | 40                 |
| 20%               | 33                 |
| 15%               | 26                 |
| 10%               | 1A                 |
| 5%                | 0D                 |
| 0%                | 00                 |

:::

```css
div {
  color: #ff0000;
} /* Красный */
div {
  color: #00ff00;
} /* Зеленый */
div {
  color: #0000ff;
} /* Синий */
```

**Пример**

> В 8‑значном hex последние два символа задают прозрачность: 0.5 → 128 в decimal → 80 в hex (можно взять значение из “Таблица цветов”)

```css
div {
  color: #00ff00; /* Зеленый */
  color: #00ff0080; /* Зеленый с прозрачностью 0.5 */
}
```

### RGB, RGBA, HSL, HSLA, HWB

::: details 2. RGB и RGBA colors

**Параметры**

| **Параметр**                | **Диапазон**          | **Описание**        |
| --------------------------- | --------------------- | ------------------- |
| **r** - red (красный)       | 0 - 255 или 0% - 100% | Интенсивность цвета |
| **g** - green (зеленый)     | 0 - 255 или 0% - 100% | Интенсивность цвета |
| **b** - blue (синий)        | 0 - 255 или 0% - 100% | Интенсивность цвета |
| **a** - alpha (альфа-канал) | 0 - 1                 |

**Примеры**

```css
div {
  /* Зеленый */
  color: rgb(0, 255, 0);
  /* Зеленый с прозрачностью 0.5 через rgba */
  color: rgba(0, 255, 0, 0.5);
  /* Зеленый с прозрачностью 0.5 через rgb (новый формат) */
  color: rgb(0 255 0 / 0.5);
}
```

:::

::: details 3. HSL и HSLA colors

**Параметры**

| **Параметр**                  | **Диапазон**                                                       |
| ----------------------------- | ------------------------------------------------------------------ |
| **hue** (чистый тон)          | 0 - 360 градусов (0 или 360 - красный; 120 - зеленый; 240 - синий) |
| **saturation** (насыщенность) | 0% - 100% (0% - оттенок серого; 100% - полный цвет)                |
| **lightness** (осветление)    | 0% - 100% (0% - черный; 100% - белый)                              |
| **alpha** (альфа-канал)       | 0 - 1                                                              |

**Варианты**

```css
div {
  color: hsl(120, 100%, 50%); /* Зеленый */
  color: hsl(120, 60%, 50%); /* Зеленый ненасыщенный */
  color: hsl(120, 100%, 25%); /* Зеленый затенённый */
  color: hsl(120, 100%, 75%); /* Зеленый осветленный */
  color: hsla(120, 100%, 50%, 0.5); /* Зеленый с прозрачностью 0.5 */
}
```

:::

::: details 4. HWB colors

**Параметры**

| **Параметр**                                          | **Диапазон**                                                       |
| ----------------------------------------------------- | ------------------------------------------------------------------ |
| **hue** (чистый тон)                                  | 0 - 360 градусов (0 или 360 - красный; 120 - зеленый; 240 - синий) |
| **whiteness** (белизна, степень «разбавления» белым)  | 0% - 100% (0% - без белого; 100% - полностью белый)                |
| **blackness** (чернота, степень «разбавления» чёрным) | 0% - 100% (0% - без чёрного; 100% - полностью чёрный)              |
| **alpha** (альфа-канал для прозрачности)              | 0 - 1                                                              |

**Варианты**

```css
div {
  color: hwb(120 0% 0%); /* Зеленый */
  color: hwb(120 50% 0%); /* Светло-зеленый */
  color: hwb(120 0% 50%); /* Темно-зеленый */
  color: hwb(120 0% 0% / 0.5); /* Зеленый с прозрачностью 0.5 */
}
```

:::

::: details 5. Predefined (стандартные цвета)

- https://drafts.csswg.org/css-color/#named-colors - Стандартные названия цветов

```css
div {
  color: red; /* Красный */
  color: green; /* Зеленый */
  color: blue; /* Синий */
}
```

:::

## Цветовые схемы

### lab, lch, oklab, oklch

::: warning
Добавить цветовые схемы `lab`, `lch`, `oklab`, `oklch`
:::

- Задание зеленого цвета

```css
div {
  color: lab(87.74% -86.18 83.18);
  color: lch(87.74% 119.78 136.02);
  color: oklab(86.64500000000001% -0.23392 0.17942);
  color: oklch(86.645% 0.29481 142.511);
}
```
