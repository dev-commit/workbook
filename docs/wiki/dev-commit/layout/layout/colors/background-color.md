# CSS свойства: background-color, color, opacity

```css
div {
  background-color: orange; /* Цвет фона */
  color: blue; /* Цвет текста*/
  opacity: 0.4; /* Задание прозрачности */
}
```

::: details `background-color` - цвет фона

```css
div {
  background-color: orange;
}
```

:::

::: details `color` - цвет текста

```css
div {
  color: blue;
}
```

:::

::: details `opacity` - задание прозрачности всему блоку и дочерним элементам

```css
div {
  opacity: 0.4;
}
```

:::
