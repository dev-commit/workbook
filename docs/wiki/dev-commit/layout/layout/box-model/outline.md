# outline

## CSS-свойства

::: details `outline` (составное свойство) - обводка

> outline часто используется на элементах формы, чтобы показать фокус при навигации с клавиатуры

**Свойства**

- `outline-width` - толщина обводки
- `outline-style` - стиль обводки
- `outline-color` - цвет обводки

**Полная форма записи**

```css
div {
  outline-width: 2px;
  outline-style: solid;
  outline-color: orange;
}
```

**Сокращенная форма записи**

```css
div {
  outline: 2px solid orange;
}
```

:::
