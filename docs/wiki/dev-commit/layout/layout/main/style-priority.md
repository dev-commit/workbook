# Приоритетность стилей

## Правила

- Правила с `!important` находятся на втором уровне каскада, в то время как обычные правила - на четвёртом
- Браузер обрабатывает селекторы справа налево

## Порядок приоритетности стилей по источнику и важности

1. Стили во время CSS-переходов (transition)
2. Браузерные стили с !important
3. Пользовательские стили с !important
4. Авторские (разработческие) стили с !important
5. Стили во время анимации
6. Обычные (т.е. без !important) авторские стили
7. Обычные пользовательские стили
8. Обычные браузерные стили
