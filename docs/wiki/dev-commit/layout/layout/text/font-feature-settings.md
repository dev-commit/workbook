# font-feature-settings

::: info

- https://ru.w3docs.com/uchebnik-css/css-svoistvo-font-feature-settings.html
  :::

> **font-feature-settings** - устранение "прыгающих" букв

## Пример

```css
div {
  font-feature-settings:
    "pnum" on,
    "lnum" on;
}
```
