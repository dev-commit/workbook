# background-clip

## CSS-свойства

### `background-clip`

::: tip `background-clip`

- **background-clip** - задает изображение на тексте
  :::

```css
span {
  outline: none;
  color: transparent;
  background: url(https://media.giphy.com/media/3o6Ztb45EYezY9x9gQ/giphy.gif);
  background-size: contain;
  background-position: top left;
  -webkit-background-clip: text;
}
```
