# RGB и RGBA colors

## Информация

| **Параметр**                | **Диапазон**          | **Описание**        |
| --------------------------- | --------------------- | ------------------- |
| **r** - red (красный)       | 0 - 255 или 0% - 100% | Интенсивность цвета |
| **g** - green (зеленый)     | 0 - 255 или 0% - 100% | Интенсивность цвета |
| **b** - blue (синий)        | 0 - 255 или 0% - 100% | Интенсивность цвета |
| **a** - alpha (альфа-канал) | 0 - 1                 |

## Примеры

```css
div {
  /* Зеленый */
  color: rgb(0, 255, 0);
  /* Зеленый с прозрачностью 0.5 через rgba */
  color: rgba(0, 255, 0, 0.5);
  /* Зеленый с прозрачностью 0.5 через rgb (новый формат) */
  color: rgb(0 255 0 / 0.5);
}
```
