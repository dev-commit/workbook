# Таблицы

## HTML-теги

- `<table />` - Контейнер
- `<tr />` - Table Row (строка)
- `<td />` - Table Data (ячейка)
- `<th />` - Table Header (ячейка заголовка)
- `<td rowspan="2" />` - Объединение 2 строк
- `<td colspan="2" />` - Объединение 2 столбцов

---

## HTML-атрибуты

- `rowspan` - объединение строки (применяется к `<td>`)
- `colspan` - объединение столбцов (применяется к `<td>`)

---

## Варианты

### Простая таблица

<table>
  <tr>
    <th>Заголовок 1</th>
    <th>Заголовок 2</th>
    <th>Заголовок 3</th>
  </tr>
  <tr>
    <td>1 строка 1 столбец</td>
    <td>1 строка 2 столбец</td>
    <td>1 строка 3 столбец</td>
  </tr>
  <tr>
    <td>2 строка 1 столбец</td>
    <td>2 строка 2 столбец</td>
    <td>2 строка 3 столбец</td>
  </tr>
</table>

### Объединение строк: rowspan

<table>
  <tr>
    <th>Заголовок 1</th>
    <th>Заголовок 2</th>
    <th>Заголовок 3</th>
  </tr>
  <tr>
    <td rowspan="2">1 строка 1 столбец</td>
    <td>1 строка 2 столбец</td>
    <td>1 строка 3 столбец</td>
  </tr>
  <tr>
    <!--<td>2 строка 1 столбец</td>-->
    <td>2 строка 2 столбец</td>
    <td>2 строка 3 столбец</td>
  </tr>
</table>
