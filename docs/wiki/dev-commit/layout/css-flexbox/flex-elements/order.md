# order

## CSS-свойства

### `order`

- Определяет порядок вывода flex-элементов
- Значение по-умолчанию: **0**

```css
.flex-element {
  /* По умолчанию у всех элементов */
  order: 0;
  /* Элемент выводится третьим по порядку */
  order: 3;
  /* Элемент выводится первым по порядку */
  order: -1;
}
```
