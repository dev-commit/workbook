# Основные свойства

## CSS-свойства

### `display`

| Значение      | Описание            |
| ------------- | ------------------- |
| `flex`        | Задание flex        |
| `inline-flex` | Задание inline-flex |

```css
.flex-container {
  display: flex;
  display: inline-flex;
}
```

### `gap`

- Задает расстояние между flex-элементами

```css
.flex-container {
  gap: 10px;
}
```
