# Поведение margin

## Центрирование элемента

- Центрирует один блок по горизонтали и вертикали

```css
.flex-element {
  margin: auto;
}
```

- Выравнивает блок по правому краю, а остальные блоки по левому

```css
.flex-element {
  margin-left: auto;
  margin-top: auto;
  margin-right: auto;
  margin-bottom: auto;
}
```
