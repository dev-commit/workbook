# grid-template-columns

## CSS-свойства

### `grid-template-columns`, `grid-template-rows`

#### Объявление

```css
div {
  /* Два столбца по 300px каждый */
  grid-template-columns: 300px 300px;

  /* Два столбца: первый на 300px, второй на 1fr - займет оставшуюся ширину */
  grid-template-columns: 300px 1fr;

  /* Две строки по 150px каждая */
  /* Если не задавать, то будут адаптироваться под высоту контента */
  grid-template-rows: 150px 150px;
}
```

#### Для grid-template-columns и grid-template-rows

**Значения: px, fr, auto**

| Значение            | Описание                                                  |
| ------------------- | --------------------------------------------------------- |
| `300px 300px 300px` | **px**. 3 элемента (фиксированные величины)               |
|                     |
| `1fr 1fr 1fr`       | **fr**. 3 элемента (относительные величины). fr - фракция |
|                     |
| `500px 1fr 2fr`     | **px + fr**. 3 элемента (смешанные величины)              |
|                     |
| `500px auto 500px`  | **px + auto**. 3 элемента. auto занимает ширину контента  |
|                     |

**Функция `repeat()`**

| Значение                                | Описание                                                                                                             |
| --------------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| `repeat(3, 1fr)`                        | **repeat()**. Задать повторяющийся паттерн                                                                           |
| `repeat(auto-fill, minmax(150px, 1fr))` | Когда минимальное значение 150px перескакивают колонки (может остаться пустое пространство сбоку)                    |
| `repeat(auto-fit, minmax(150px, 1fr))`  | Когда минимальное значение 150px перескакивают колонки (пустое пространство не оставляет, блок заполняет всю ширину) |

**Функция `minmax()`**

| Значение                       | Описание                                                          |
| ------------------------------ | ----------------------------------------------------------------- |
| `minmax(150px, 250px) 1fr 1fr` | **minmax() px**. 3 элемента, первый с minmax                      |
|                                |
| `minmax(100px, auto)`          | **minmax() auto**. Определяется автоматически по ширине элементов |
|                                |

### `grid-template`

- Короткая запись строки / столбцы

```css
div {
  grid-template: 150px 150px / 300px 300px;
}
```
