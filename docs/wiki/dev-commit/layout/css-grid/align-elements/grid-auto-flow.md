# grid-auto-flow

## CSS-свойства

### `grid-auto-flow`

::: info

- https://developer.mozilla.org/ru/docs/Web/CSS/Reference/Properties/grid-auto-flow - MDN
  :::

| Значение | Описание                                               |
| -------- | ------------------------------------------------------ |
| `row`    | Порядок элементов по очереди по строкам (по умолчанию) |
| `column` | Порядок элементов по очереди по столбцам               |
| `dense`  | Заполняет пустые места                                 |

```css
div {
  grid-auto-flow: row;
}
```
