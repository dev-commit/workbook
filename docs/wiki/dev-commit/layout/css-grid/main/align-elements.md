# CSS-свойства для выравнивания элементов

## CSS-свойства

### `justify-items`, `align-items`, `justify-self`, `align-self`

- `justify-items` - у родительского элемента (горизонтальное)
- `align-items` - у родительского элемента (вертикальное)
- `justify-self` - у дочернего элемента (горизонтальное)
- `align-self` - у дочернего элемента (вертикальное)

| Значение | Описание                                                            |
| -------- | ------------------------------------------------------------------- |
| stretch  | Блоки растягиваются по ширине (по умолчанию)                        |
| start    | По левому краю. блоки не растягиваются (только по ширине контента)  |
| end      | По правому краю. блоки не растягиваются (только по ширине контента) |
| center   | По центру. блоки не растягиваются (только по ширине контента)       |

```css
div {
  justify-items: stretch;
  align-items: stretch;
  justify-self: stretch;
  align-self: stretch;
}
```
