# Свойства и атрибуты

## CSS-свойства

### `fill`, `stroke`

```css
svg {
  /* Заливка */
  fill: yellowgreen; /* Цвет заливки */
  fill-opacity: 0.9; /* Прозрачность заливки */

  /* Линия */
  stroke: black; /* Цвет линии */
  stroke-width: 2; /* Толщина линии */
  stroke-opacity: 0.7; /* Прозрачность линии */
  stroke-dasharray: 2 2; /* Пунктирная линия */

  /* Края линии */
  stroke-linecap: butt; /* Прямоугольник (default) */
  stroke-linecap: square; /* Расширенный прямоугольник */
  stroke-linecap: round; /* Скругление */
}

svg path {
  d: path("M0,100 L 20,0 80,90 100,0 100,100 Z");
}
```

- `stroke-linecap: round` - в конце линии обводка будет расширена прямоугольником с шириной, равной половине ширины обводки, и высотой, равной ширине обводки

## Теги и атрибуты

### `<g>`

- Тег `<g></g>` - объединение объектов для задания общих свойств
- Атрибут `preserveAspectRatio="none"` позволяет поместить фигуру целиком в viewBox. При изменении размера блока svg размер фигуры будет меняться
