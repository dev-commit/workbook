# @import

> **@import** - подключения стилей. Подключается в начале файла стилей

## Варианты

### Подключение файла стилей

```css
@import "main.css";
@import "main.css" screen;
```

### Подключение шрифта

```css
@import url(https://fonts.googleapis.com/css?family=Oswald);

span {
  font-family: 'Oswald', sans-serif;'
}
```
