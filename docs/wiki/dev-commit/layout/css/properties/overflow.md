# overflow

## CSS-свойства

### overflow

::: details `overflow` - отображение содержимого элемента

- Свойство overflow управляет отображением содержимого блочного элемента, если оно целиком не помещается и выходит за область заданных размеров

| Значение | Описание                                                      |
| -------- | ------------------------------------------------------------- |
| `auto`   | Добавление полос прокрутки, если контент не помешается в блок |
| `hidden` | Обрезка контента, если контент не помешается в блок           |

```css
div {
  overflow: auto;
  overflow: hidden;
}
```

:::
