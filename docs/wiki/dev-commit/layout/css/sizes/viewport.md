# Единицы измерения: viewport

### vm и vh

> **vw** **vh** - эквивалентно 1% ширины (vw) / высоты (vh) окна браузера

- vw (viewport width) - эквивалентно 1% ширины окна браузера
- vh (viewport height) - эквивалентно 1% высоты окна браузера
- Единицы измерения, которые рассчитываются относительно viewport
- Включают в себя ширину скроллбара

**Пример**

- Высота окна браузера 600px → 1vh = 6px
- Ширина окна браузера 1200px → 1vw = 12px

**Пример. Задание у блока 100% ширины и высоты (в vw и vh)**

```css
div {
  /* Ширина блока 100% от viewport */
  width: 100vw;
  /* Высота блока 100% от viewport */
  height: 100vh;
}
```

### vmin и vmax

> **vmin** **vmax** - эквивалентно 1% меньшего (vmin) / большего (vmax) размера окна браузера по высоте или ширине

- vmin - эквивалентно 1% меньшего размера окна браузера по высоте или ширине
- vmax - эквивалентно 1% большего размера окна браузера по высоте или ширине

**Пример**

- Ширина окна браузера 1200px
- Высота окна браузера 600px
- 1vmin = 6px
- 1vmax = 12px

```css
div {
  width: 100vmin;
  height: 100vmin;
}
```
