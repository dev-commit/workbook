# Единицы измерения: px и %

## px

> **px** - абсолютная единица измерения

```css
div {
  width: 100px;
  height: 100px;
}
```

## Проценты %

> "%"" - относительная единица измерения. Рассчитывается относительно родителя

### Пример. Задание у блока 100% ширины и высоты (в процентах)

```css
html,
body {
  height: 100%;
}
div {
  width: 100%;
  height: 100%;
}
```
