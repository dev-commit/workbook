# Информация

## Абсолютные и относительные единицы измерения

::: tip Определения

- **Абсолютная единица измерения** - фиксированная единица, которая всегда представляет одинаковый физический размер, независимо от контекста или размера устройства. Эти единицы не зависят от размеров родительских элементов или экрана.
- **Относительная единица измерения** - единица, значение которой зависит от контекста или других параметров, таких как размер родительского элемента или базовый размер шрифта.
  :::
