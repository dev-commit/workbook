# Псевдоклассы

## Варианты

### Пустые элементы и отрицание

| Псевдокласс       | Описание                       |
| ----------------- | ------------------------------ |
| `p:empty`         | все пустые элементы p          |
| `p:not(:empty)`   | все не пустые элементы p       |
| `p:not(.myclass)` | если p не имеет класса myclass |

### Нумерация и дочерние элементы

| Псевдокласс           | Описание                                                                                         |
| --------------------- | ------------------------------------------------------------------------------------------------ |
| `div:nth-of-type(2n)` | каждый второй элемент определённого типа                                                         |
| `div:only-child`      | только дочерние div                                                                              |
| `:nth-last-child`     | стиль по нумерации в дереве (отсчёт от последнего элемента, в отличие от :nth-child)             |
| `:nth-last-of-type`   | стиль к элементам указанного типа по нумерации (отсчёт от последнего, в отличие от :nth-of-type) |

### Состояния полей формы

| Псевдокласс          | Описание                                                                              |
| -------------------- | ------------------------------------------------------------------------------------- |
| `input:checked`      | выбранный элемент для checkbox и radio                                                |
| `input:enabled`      | активные элементы                                                                     |
| `input:disabled`     | неактивные элементы                                                                   |
| `input:invalid`      | поля, содержимое которых не соответствует типу (напр. незаполненные input с required) |
| `input:valid`        | поля, содержимое которых соответствует указанному типу                                |
| `input:optional`     | input без required                                                                    |
| `input:required`     | input с required                                                                      |
| `input:out-of-range` | превышение лимита max в input number                                                  |
| `input:in-range`     | в пределах min–max в input number                                                     |
| `input:read-only`    | input с атрибутом readonly                                                            |
| `input:read-write`   | input, доступные для изменения                                                        |

### Ссылки, тип элемента и язык

| Псевдокласс         | Описание                                                                                           |
| ------------------- | -------------------------------------------------------------------------------------------------- |
| `div:target`        | стилизация элемента с id при переходе по ссылке (напр. `<a href="#target1">` → элемент `#target1`) |
| `div:first-of-type` | первый элемент определённого типа                                                                  |
| `div:last-of-type`  | последний элемент определённого типа                                                               |
| `div:only-of-type`  | дочерний элемент указанного типа, если он единственный у родителя                                  |
| `div:lang`          | язык, используемый в документе                                                                     |

### Специальные состояния

| Псевдокласс      | Описание                                                |
| ---------------- | ------------------------------------------------------- |
| `:root`          | корневой элемент документа                              |
| `:default`       | элемент по умолчанию (напр. одна из двух кнопок submit) |
| `:fullscreen`    | полноэкранный режим                                     |
| `:indeterminate` | переключатель в неопределённом состоянии                |
