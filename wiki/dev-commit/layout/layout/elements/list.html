CSS-свойства ​

Настройка маркеров ​

list-style (составное свойство) - маркеры

list-style-type - изменение вида маркера
css
ul {
  list-style-type: none;
  list-style-type: circle;
  ...
}
list-style-image - добавление изображения вместо стандартного маркера
css
ul {
  list-style-image: none;
  list-style-image: url("Путь к файлу");
}
list-style-position - задание позиции относительно текста
css
ul {
  list-style-position: inside;
  list-style-position: outside;
}

Формы записи ​

Полная форма записи

css
ul {
  list-style-type: none;
  list-style-image: url("Путь к файлу");
  list-style-position: inside;
}

Сокращенная форма записи

css
ul {
  list-style: none url("Путь к файлу") inside;
}
