## 🪲 Bugs

### Ломается новый Layout при перезагрузке страницы

- **Ошибка**: При перезагрузке страницы или переходе по прямой ссылке перестает отрображаться часть данных: заголовки, блоки с кодом. Это просходит только на build приложения, размещенном на GitHub Pages, на локально запущенном приложении такой проблемы нет

- **Где воспроизводится**: только на новом Layout из "docs/.vitepress/theme/components/Layout.vue". На стадратном layout VitePress и на мобильной версии работает корректно

- **На каких сраницах воспроизводится**: на страницах где есть любой блок кода в "``` ```", возможно, в каких-то случаях и в "`code`" (нужно проверить)
